full outer join

거르는 거 없이 모든 데이터 합치기

```
select user.user_id, user.name , addr.addr
from user
left join addr
on user.user_id = addr.user_id
union
select addr.user_id, user.name , addr.addr
from user
right join addr
on user.user_id = addr.user_id;
```
